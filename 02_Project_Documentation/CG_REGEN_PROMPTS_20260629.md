## 📋 生成后导入指南

1. 在Midjourney中对每张提示词生成图片（建议生成4张变体后选择最佳的Upscale）
2. Upscale后点击图片在浏览器中打开原图（确保是完整分辨率）
3. 右键保存图片，**严格按照上表中的文件名命名**（如 `cg_yewenjie_signal.png`）
4. 将所有保存的PNG文件放入 `03_Web_Rebuild/public/images/` 目录覆盖原文件
5. 运行 `node scripts/verify-assets.mjs` 验证文件大小（高质量CG应大于3MB）
6. 运行 `npm run build` 重新构建并验证

> **注意**：Midjourney生成的是WebP或JPEG，请在保存时确保转换/导出为PNG格式，或者直接保存PNG（使用Midjourney的下载原图功能）。
