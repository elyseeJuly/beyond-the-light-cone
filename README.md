## 🌌 游戏简介

《宇宙群英传》是一款以刘慈欣《三体》系列小说为背景的深度回合制策略游戏。你将在2009年危机纪元中临危受命，担任地球联合政府最高执政官，面对三体舰队的威胁和多文明并存的黑暗森林宇宙。

**你的每一个决策，都将决定人类文明的最终命运。**

## 🎮 核心游戏系统

### 🏭 经济与资源生产链

游戏拥有一套完整的四维生产经济系统，与《三体》世界观深度融合：

| 系统 | 机制 | 三体关联 |
|------|------|----------|
| **采矿系统** | 工人分配矿厂开采星球资源，受科技权重(星矿Ⅰ→Ⅲ)和部长加成影响 | 逃亡主义会削弱产出，对应原著中社会动荡对生产力的破坏 |
| **工厂系统** | 资源→经济转换 (2:1)，行星发动机技术可提升5倍产能 | 质能转换科技可免除资源消耗，致敬原著中的科技奇迹 |
| **文化系统** | 文化工人产出文化值，思想钢印科技提升权重 | 文化值推动文明等级进化，思想钢印可选强化或纯粹路线 |
| **人口增长** | 每座太空城市提供人口增长，殖民城科技加速增长率 | 人力资源部长可提供额外加成，面壁者/执剑人影响军力 |

**工人分配**：你可以在采矿(30%) / 工厂(30%) / 文化(30%) 之间调配闲置劳动力，每个决策都有明显的策略分化。

**逃亡主义**：每回合随机上升 0~2 点，达到 100 即游戏失败。面壁者和执剑人可以通过提升军力来延缓社会分裂。

### 🔬 科技树（85 节点 × 5 条线）

完整复刻 MFC 原版全部科技节点，每条线都有具体的数值影响：

| 科技树 | 节点数 | 核心路线 | 终极目标 |
|--------|--------|----------|----------|
| **物理学** | 23 | 天文观测→50光年/1万光年/银河系远镜；粒子对撞→质子展开→智子工程；维度物理→曲率驱动→光速飞船 | 黑域生成（黑域胜利） |
| **航天工程** | 37 | 化学推进→10%/50%/99%/光速飞船；核聚变→行星发动机Ⅰ→Ⅱ→Ⅲ；行星工程→地下城Ⅰ→Ⅱ→Ⅲ | 行星发动机Ⅲ型 + 新家园选址（流浪胜利） |
| **军事武器** | 13 | 小行星级→行星级→恒星级氢弹；宏原子聚变→球状闪电→宏化部队；黑暗森林威慑→引力波广播→万有引力号 | 二向箔武器化 |
| **信息技术** | 14 | 思想钢印Ⅰ→Ⅱ→Ⅲ（文化权重）；数字文明→意识上传→550W量子计算机→数字方舟 | 数字方舟（数字永生胜利） |
| **星际文明** | 12 | 宇宙社会学→猜疑链→技术爆炸预判；流浪地球计划→新家园选址 | 银河共同体 / 宇宙重启 |

每条科技线都有确切的数值收益：星矿权重 2→5、工厂权重 2→5、殖民城权重 2→5、思想钢印权重 2→5。科技不再是"为研究而研究"。

### ⚔️ 军事与战斗

采用 **5 轮骰子制** 战斗系统，告别简单数值比拼：

- **攻击骰子**：0.8 + Math.random() × 0.4，随轮次递增 +10% 伤害
- **防御骰子**：0.75 + Math.random() × 0.5
- **武器区分**：水滴/探测器 ×20、恒星级战舰 ×15、常规武器 ×10
- **将领加成**：army × 0.1 + leadership × 0.05（进攻），army × 0.15（防守）
- **僵局判定**：5 轮后剩余 HP 比 > 1.3 进攻胜，否则防守胜

### 🌍 异星文明 AI（5 种人格）

| 人格 | 代表文明 | 行为模式 |
|------|----------|----------|
| **猎手型** (Hunter) | 三体 | 18% 概率攻击，5-10 回合冷却，威慑纪元/广播纪元时 30% 跳过 |
| **清理者型** (Cleaner) | 歌者 | 12% 概率攻击，3-7 回合冷却，威慑 > 70 时跳过 |
| **扩张型** (Expansionist) | 边缘世界 | 14% 概率殖民无人星球，威慑 < 80 方可扩张 |
| **防御型** (Defensive) | 魔戒 / 归零者 | 仅 +5 军力/回合，极少主动出击 |
| **机会型** (Opportunist) | — | 友好时 8% 索取经济援助，威慑 < 50 时 25% 进攻 |

### 🤝 外交系统

完整的五级关系链：愤怒 → 不悦 → 中立 → 友好 → 亲密

| 行动 | 效果 | 代价 |
|------|------|------|
| 谈判 | 关系 +1 | 3 回合冷却 |
| 贸易 | 经济 -30，资源 +50 | 3 回合冷却 |
| 挑衅 | 关系 -1 | 3 回合冷却 |
| 结盟 | 设为同盟，停止攻击 | 需友好以上 |

### 📜 事件叙事系统

三层事件引擎：

1. **固定年份事件**：关键年份触发（年 0/2/10/50/300），推动主线剧情，并解锁特定的历史人物加入阵营（如罗辑、章北海、程心等）。支持严格三体模式过滤（如屏蔽流浪地球彩蛋）。
2. **随机事件池**：内置 200KB+ 丰富随机事件，包含危机、威慑直到银河纪元全覆盖。支持多纪元限定与逻辑白名单拦截。系统级内置 **50% 数值溢出钳制**，确保大后期数据不崩盘。
3. **条件过滤事件**（预置故事线）：
   - 面壁者选拔（年 10-50，文化 ≥ 30）
   - 威慑体系建立（需黑暗森林威慑科技，威慑 ≥ 50）
   - 智子封锁生效（年 10-200，科技 -90%）
   - 流浪地球大辩论（年 100+，需行星发动机基础）
   - 归零者广播与暗物质海微观文明遭遇（需进入银河纪元）
   - 逃亡主义叛乱（年 60+，逃亡 ≤ 30）
   - 智子反制突破（需 550W 量子计算机）

**官方时间线对比系统**：内置完整的《三体》宇宙官方纪元时间线。游戏内提供「编年史观测」随时对比进度，当游戏结局时将展示双轨时间线（官方原著 vs 玩家本局真实履历），让您复盘历史的蝴蝶效应。

**智能角色指派**：在指派各部门负责人与面壁者时，系统会根据岗位核心需求自动按能力值降序排列人才，最适合的人选永远置顶。

**Flag 持久化系统**：事件选项会设置 Flag，后续事件依据 Flag 触发不同分支，形成因果链。

### 🏆 胜利条件（6 胜 + 3 败）

**胜利条件**：
| 胜利类型 | 达成条件 |
|----------|----------|
| 流浪胜利 | 行星发动机 Ⅲ 型 + 新家园选址 |
| 数字永生胜利 | 完成数字方舟 |
| 威慑胜利 | 威慑纪元 + 执剑人存活 |
| 征服胜利 | 消灭或同盟所有异星文明 |
| 黑域胜利 | 完成黑域生成技术 |
| 隐藏胜利 | 宇宙重启理论 |

**失败条件**：
- 逃亡主义 ≥ 100（社会崩解）
- 人口 ≤ 0（文明灭绝）
- 年 > 400 且纪元 < 银河纪元（太阳氦闪）

### ⏳ 纪元系统

年 1-200：**危机纪元** → 年 201-260：**威慑纪元** → 年 261-268：**广播纪元** → 年 269-330：**掩体纪元** → 年 331+：**银河纪元**

### 👑 文明等级

文化值累积推动文明进化：荒蛮 → 起源（70） → 风暴（200） → 逐鹿（500） → 霸王（1000），每次升级 +20 军力。

### 🔄 新游戏+ (New Game+) 继承系统

为了彻底打通游戏循环闭环，Web 重构版实现了全新的 **新游戏+ (New Game+)** 系统：
- **结局存档与成就树**：每次达成 9 种胜利/失败结局之一时，系统会自动将对应的结局记录持久化到 `localStorage` 的结局成就库。
- **开局继承奖励**：重新启动游戏时，新游戏将继承历史解锁成就的红利加成。玩家能获得额外的初始经济、矿产资源储备，或提高基础威慑度、调低初始社会逃亡主义倾向，为应对三体危机和漫漫黑夜提供更深厚的底蕴支撑。

### 🎵 音频与声音管理系统

游戏配备了高还原度且适应浏览器策略的音频架构：
- **独立音频管理器 (AudioManager)**：基于 HTML5 **Web Audio API 振荡器 (Oscillator)**。在无需加载任何外部音效文件的前提下，通过振荡器实时合成锯齿波警报声、正弦波里程碑达成和弦以及负面事件的失调音效，极大地减小了前端资源开销。
- **BGM 交互式自动播放**：游戏背景音乐《岁月底座》支持自动循环播放。若由于浏览器的自动播放限制（Autoplay Policy）被拦截，系统会在玩家与页面进行首次交互（点击或按键）时，自动静默唤醒并恢复播放。
- **沉浸式结局音乐流**：大结局主题曲《星屑》移至结局场景顶层加载，自结局宣告（Phase 1）伊始即开始无缝循环播放，贯穿整个场景演绎、编年史回顾及制作人员名单滚动流程，营造完整的叙事共鸣与感官冲击力。

## 🚀 Web 重构版

### 技术栈

| 层级 | 技术 |
|------|------|
| 框架 | React 19 |
| 语言 | TypeScript 5 (strict 模式) |
| 构建 | Vite 8 |
| 样式 | Tailwind CSS 4 |
| 图标 | Lucide React |
| 动画 | Framer Motion |
| 渲染 | HTML5 Canvas 2D |
| 存档 | localStorage (Map/Set 序列化) |

### 快速启动

```bash
cd 03_Web_Rebuild
npm install
npm run dev        # 启动开发服务器
npm run build      # 生产构建
npm run preview    # 预览构建产物
```

### Windows vs Web 主要区别

| 特性 | Windows 原版 (2009) | Web 重构版 (2026) |
|------|---------------------|-------------------|
| 语言 | C++ (MFC) | TypeScript (React) |
| 图形 | DirectX 9 / GDI | Canvas 2D / CSS3 |
| 存储 | 二进制 .sav / INI | JSON / localStorage |
| 平台 | 仅 Windows | 全平台浏览器 |
| 编码 | GBK | UTF-8 |
| 科技树 | 74 节点 | 85 节点（完整复刻） |
| 星系渲染 | 随机分布星空 | 严格对数真实光年轨道距离 |
| 战斗 | 单次数值比较 | 5 轮骰子制与军队裁剪 |
| AI | 统一 1/7 概率 | 5 种人格差异化 |
| 事件 | 年份触发 | 三层引擎 + Flag 因果链 + 防数值溢出钳制 |
| 胜利条件 | 4 种 | 6 胜 + 3 败 |
| 存档与继承 | 单局独立，无继承 | 支持 New Game+ 继承系统（基于成就和结局历史提供初始加成） |
| 音乐与音效 | DirectShow 音乐播放 | 独立音频管理器 (Web Audio API 实时合成音效 + BGM 交互式自动播放 + 沉浸式结局音乐) |
